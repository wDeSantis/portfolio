Jeu Brick-Breaker

Ce jeu est un projet académique que j'ai créer où les joueurs peuvent accéder à différents niveaux. Le joueur contrôle une palette pour diriger une balle et détruire un ensemble de briques. À chaque niveau le nombre de briques augmente ainsi que la difficulté.

Réalisé avec: HTML, CSS, Javascript
